Dasar-dasar dari bagaimana menggunakan git Hasil pembelajaran -Dengan Akhir ini Anda harus dapat: perintah git -Jalankan dari baris perintah -Nama Perintah git digunakan untuk mendapatkan salinan lengkap sebuah repositori git ada dari Github -Nama Perintah git digunakan untuk memeriksa status file Anda -Nama Perintah git digunakan untuk melacak file -Nama Perintah git digunakan untuk melakukan file -Nama Perintah git digunakan untuk melihat Anda sejarah commit -Nama Perintah git digunakan untuk meng-upload proyek ke GitHub Sebuah Cheatsheet Berikut adalah daftar referensi dari perintah git paling umum digunakan (Anda mungkin mempertimbangkan bookmark halaman web ini berguna). Cobalah untuk membiasakan diri dengan perintah dan kemudian akhirnya ingat mereka: * Perintah yang berhubungan dengan repositori remote: () Git clone https://github.com/user-name/repository-name.git () Master git push origin * Perintah yang berhubungan dengan alur kerja: () Git add. () Git commit -m "Sebuah pesan yang menjelaskan apa yang telah Anda lakukan untuk membuat snapshot ini berbeda" * Perintah yang berhubungan dengan memeriksa status atau log sejarah () Git status () Log git Sintaks git bekerja seperti ini: Sebagai contoh: () Git add. (Dibaca seperti) git | tambahkan | . () Git commit -m "pesan" (dibaca seperti git) | komit -m | "pesan" () Git status (dibaca seperti) git | status | (Tanpa tujuan)
